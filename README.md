## About The Project
This is C console program that can calculate area and volume of a rectangle and cuboid. v1.0

## Installation
Download [code2-1.0.tar.gz](https://github.com/jilquero/Code2/raw/main/code2-1.0.tar.gz).

`tar zxvf code2-1.0.tar.gz`

`cd code2-1.0`

`./configure`

`make`

`sudo make install`
